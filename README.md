LILLY MILK SYSTEM

📌 Project Overview
Lilly Milk System is a requirement analysis project for a milk ordering and delivery system.  
The project focuses on business analysis, system design, and UI/UX planning.

🎯 Objectives
- Analyze business requirements
- Define system features and user roles
- Design system workflows and diagrams
- Create UI/UX prototype using Figma

👥 User Roles
- Customer: place and track orders
- Admin: manage products and orders
- Staff: handle delivery and processing
